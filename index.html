Minha Biblioteca: Uma Webpage Personalizada

O projeto é uma página web desenvolvida utilizando HTML e CSS, com o objetivo de apresentar um portfólio pessoal. A proposta é criar um espaço online onde seja possível expor informações sobre habilidades, projetos realizados e formas de contato, funcionando como uma vitrine profissional na web.

A estrutura do site foi construída com HTML, que organiza a disposição e o significado do conteúdo, enquanto o CSS foi aplicado para definir a aparência visual da página, ajustando cores,fontes e a disposição dos elementos para tornar a navegação agradável e intuitiva.

O HTML organiza o conteúdo em uma estrutura clara, composta por um cabeçalho com o título e links de navegação, uma seção de apresentação pessoal, uma galeria de projetos e uma área de contato. Cada projeto é destacado com uma imagem representativa, uma breve descrição e um link para mais detalhes. O design responsivo é assegurado através do uso de media queries e de uma configuração adequada da viewport, permitindo que o site se adapte bem a diferentes tamanhos de tela.

O CSS é responsável pela estilização completa do site, utilizando variáveis para definir as cores principais, fontes modernas importadas do Google Fonts, além de efeitos de transição em botões e elementos interativos. A combinação de HTML e CSS permite criar um portfólio visualmente atrativo, acessível e funcional, sem necessidade de linguagens adicionais, garantindo uma experiência agradável para quem deseja conhecer melhor os trabalhos e habilidades apresentados.

Minha Biblioteca: Uma Webpage Personalizada

O projeto é uma página web desenvolvida utilizando HTML e CSS, com o objetivo de proporcionar uma experiência interativa baseada em uma narrativa de aventura. A proposta é criar um ambiente digital onde os usuários possam explorar uma floresta misteriosa, tomando decisões que influenciam o desenrolar da história, semelhante aos clássicos livros-jogo.

A estrutura do site foi construída com HTML, que organiza a disposição e o significado de cada parte do conteúdo, enquanto o CSS é aplicado para definir a estética da página, utilizando cores que remetem à natureza, fontes temáticas e uma disposição visual que facilita a imersão na aventura.

O HTML organiza o conteúdo de forma lógica e clara, começando com um cabeçalho que apresenta o título da aventura, seguido por seções que descrevem o enredo e oferecem opções de escolha ao usuário, guiando-o através de diferentes caminhos na floresta. Cada escolha leva a novas seções ou páginas, criando um fluxo dinâmico e envolvente. O design responsivo é garantido por meio da configuração da viewport e do uso de técnicas de layout flexível, assegurando que a experiência seja acessível em diversos dispositivos e tamanhos de tela.

O CSS é responsável por toda a estilização do site, utilizando paletas de cores naturais e fontes importadas para reforçar a ambientação da floresta. Além disso, são aplicados efeitos visuais suaves para destacar as escolhas interativas e tornar a navegação mais envolvente. A combinação de HTML e CSS neste projeto é fundamental para oferecer uma experiência de usuário acessível, funcional e esteticamente atrativa, promovendo a interação e estimulando a exploração do conteúdo narrativo de forma intuitiva e criativa.
